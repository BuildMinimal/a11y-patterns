WCAG 2.4.1 — Bypass Blocks

Category: Navigable · Level: A

A mechanism is available to bypass blocks of content that are repeated on multiple web pages. For keyboard-only users, navigating past a large navigation menu on every page visit is a significant burden. A skip link solves this in one Tab press.

Screen reader users have their own bypass mechanisms — heading navigation, landmark regions, links list. But sighted keyboard users have none of these unless the page explicitly provides a skip link.
